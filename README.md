Homebrew repository to install Umschlag on macOS.


## Prepare

```bash
brew tap umschlag/umschlag
```


## Install

### umschlag-cli

```bash
brew install umschlag-cli
umschlag-cli -h
```

### umschlag-api

```bash
brew install umschlag-api
umschlag-api -h
```

### umschlag-ui

```bash
brew install umschlag-ui
umschlag-ui -h
```
